Capabilities

What this provider supports

Everything you need for a production-grade auth server — built from scratch.

🔐

OAuth 2.0 Authorization Code Flow

Implements secure OAuth login using authorization codes, redirect URIs, client validation, and token exchange endpoints.

🪪

OpenID Connect Support

Supports OIDC discovery, identity claims, ID tokens, and UserInfo endpoint for authentication and user identity.

🔑

RS256 JWT Authentication

Tokens are cryptographically signed using RSA private/public key pairs and verified through /.well-known/jwks.json.

♻️

Secure Refresh Token Flow

Refresh tokens are hashed and stored in the database. Short-lived access tokens (1h) protect APIs without re-login.

🌐

Single Sign-On (SSO)

Session-based SSO across multiple client apps. Log in once, skip login on every other registered client automatically.

📋

Client Registration

Register client apps dynamically via /o/register-client. Each app gets a unique client_id and hashed client_secret.
